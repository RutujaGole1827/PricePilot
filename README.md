# price_pilot

This system is a Pricing Management Tool for a company that sells food and beverages at places like airports, stations, or travel locations.
What it does?
It helps company staff for:
Set product prices
Update prices across different stores/locations
Check profit margins
Compare old vs new prices
Send price changes for approval
Track who changed what
Generate pricing reports
Example:
If the price of coffee increases in one airport:
A manager updates the new price in this tool
Finance checks profitability
Another manager approves it
The updated price is applied to selected stores
So basically, it is a central system to control and manage pricing operations efficiently across many outlets.
